<h1> Posta aí: Aplicação Web com MongoDB e Tailwind CSS</h1> 

![website screenshot](https://github.com/user-attachments/assets/cbf2a641-0458-4d87-b3a5-caf9e83c8ec6)

A ideia inicial surgiu ao realizar um trabalho de faculdade, então o projeto evoluiu para uma aplicação web mais robusta, utilizando o MongoDB como banco de dados e integrando o framework Tailwind CSS para uma estilização moderna.

<h3>Status: </h3> 
<img width="110px" src="https://i.imgur.com/n8nELsw.png">
<h4>Features: </h4>
• Criar post-its com texto (ideia inicial)<br>
• Criar post-its com desenhos<br>
• Visualizar post-its de toda comunidade ✔<br>
• Compartilhar post-its ✔<br>


<h3> Tecnologias Utilizadas</h3> 
• React<br>
• Node.js<br>
• Express<br>
• MongoDB<br>
• Tailwind CSS<br>
• Cors<br>
• Nodemon<br>
